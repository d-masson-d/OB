Чек-лист тестирования сайта колледжа НКЭиВТ (nke.ru)

1. Онлайн-приёмная (форма обращения)

Доступность и структура
- Форма открывается без ошибок
- Присутствуют все обязательные поля (ФИО, контакты, тема, текст обращения)
- Есть текст согласия на обработку персональных данных
- Подписи к полям понятны и не противоречат плейсхолдерам

Валидация данных
- Пустые обязательные поля вызывают сообщения об ошибке
- ФИО не принимает цифры и спецсимволы
- E-mail проверяется на корректность формата
- Телефон принимает только допустимые форматы
- Ограничена максимальная длина текста обращения
- HTML / JS-код в полях не исполняется (защита от XSS)
- Без согласия на обработку данных форму отправить нельзя

Кнопка отправки и UX
- Кнопка «Отправить» не отправляет форму при ошибках
- Ошибки отображаются понятно и рядом с полем
- Введённые данные не пропадают при ошибке
- После отправки появляется подтверждающее сообщение
- Возможно навигация по форме с клавиатуры (Tab)

2. Поиск по сайту

Общая работа поиска
- Поле поиска отображается и доступно
- Поиск запускается кнопкой и клавишей Enter
- Отображается страница результатов поиска
- Корректно выводится сообщение «Ничего не найдено»

Типы поисковых запросов
- Одно слово / несколько слов
- Разный регистр букв (ПОИСК / поиск)
- Запросы с цифрами
- Запросы с опечатками
- Запросы со спецсимволами
- Пустой запрос
- Очень длинный запрос

Качество и навигация
- Результаты соответствуют запросу
- В выдаче есть новости, документы и основные разделы
- Нет дубликатов страниц
- Работает переход по страницам (пагинация)
- Поиск не выдаёт служебные или закрытые страницы

3. Проверка шапки сайта (header)

Содержимое и навигация
- Логотип отображается корректно
- Клик по логотипу ведёт на главную страницу
- Все пункты меню видны и кликабельны
- Активный раздел выделяется
- Выпадающие меню работают корректно

Функциональные элементы
- Поле поиска в шапке работает
- Кнопка «Онлайн приёмная» ведёт на форму
- Контактная информация отображается
- Ссылки на соцсети открываются корректно

Адаптивность
- Меню корректно отображается на мобильных устройствах
- Бургер-меню открывается и закрывается без ошибок
- Элементы шапки не перекрывают друг друга

4. Главная страница сайта

Загрузка и отображение
- Главная страница открывается без ошибок
- Контент отображается без поломки верстки
- Есть основной заголовок страницы (h1)

Контент и блоки
- Присутствует информация о колледже
- Отображаются актуальные новости / объявления
- Есть блоки для абитуриентов и студентов
- Все кнопки «Подробнее» ведут на правильные страницы

Ссылки и медиа
- Все внутренние ссылки работают
- Документы открываются или скачиваются корректно
- Изображения загружаются без ошибок
- Слайдеры и баннеры работают стабильно

Адаптивность
- Страница корректно отображается на телефонах
- Нет горизонтальной прокрутки
- Элементы удобны для нажатия

5. Возможные баги

- Битые ссылки на документы или старые страницы
- Неактуальная информация на главной
- Ошибки 404 / 500 при переходе по разделам
- Проблемы с меню на мобильных устройствах
- Потеря данных в форме при ошибке
- Поиск не находит очевидные разделы сайта
- Отсутствие alt-текста у изображений
