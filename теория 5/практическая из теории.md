Практическая работа: Классификация тестирования

Объект исследования: Наручные электронные часы (смарт-часы / фитнес-браслет)

Выполнил: Стрепетов Д.О.
Группа: 9ИС - 420
Предмет: Обеспечение качества

Таблица видов тестирования

| Вид тестирования | Пример тестовой проверки |
| Функциональное | Проверить, корректно ли отображается текущее время после синхронизации со спутником/телефоном |
| Регрессионное | Убедиться, что после обновления прошивки по-прежнему работает подсчет шагов и пульсометр |
| Нагрузочное | Запустить одновременно трекинг GPS, музыкальный плеер и уведомления — проверить, не зависнет ли система и на сколько хватит батареи |
| Интеграционное | Проверить сопряжение часов со смартфоном (разные модели iPhone и Android) и корректность передачи данных |
| Модульное (компонентное) | Проверить работу отдельно взятого датчика пульса без включения остальных функций |
| Тестирование удобства использования (Usability) | Оценить, насколько удобно пролистывать меню мокрыми пальцами во время бега |
| Тестирование безопасности (Security) | Проверить, можно ли получить доступ к данным часов через чужой телефон без подтверждения пароля |
| Тестирование совместимости (Compatibility) | Проверить отображение уведомлений из разных приложений (WhatsApp, Telegram, SMS) на экране часов |
| Тестирование установки (Installation) | Убедиться, что мобильное приложение для часов корректно устанавливается и удаляется без "мусора" в системе |
| Дымовое (Smoke test) | Включить часы, проверить, загружается ли главный экран и реагирует ли сенсор на нажатие |
| Тестирование локализации | Проверить, корректно ли отображаются дни недели и даты на русском и английском языках |
| Тестирование доступности (Accessibility) | Проверить, можно ли пользоваться часами людям с ослабленным зрением (регулировка яркости, крупный шрифт) |
| Деструктивное тестирование | Проверить, выдержат ли часы падение на бетонный пол с высоты 1 метр |
